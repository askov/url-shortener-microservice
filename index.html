Free code camp: URL Shortener Microservice

User story:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • When I visit that shortened URL, it will redirect me to my original link.

Usage

Create short URL

Shortened URL for your link can be created using the following scheme:

https://url-shortener-microservice-aa.glitch.me + /new/ + 〈 your_url 〉

If the form of provided URL is valid response will be:

{original_url: 〈 original_url 〉, short_url: 〈 short_url 〉}

Invalid URLs will lead to an error:

{error: 〈 error_message 〉}

Use provided shortened URL

The short link received will redirect you to original URL.

https://url-shortener-microservice-aa.glitch.me + / + 〈 short_url 〉

Examples

  • https://url-shortener-microservice-aa.glitch.me/new/https://google.com
  • https://url-shortener-microservice-aa.glitch.me/new/https://ru.wikipedia.org/wiki/Yahoo!
  • https://url-shortener-microservice-aa.glitch.me/new/https://golang.org/pkg/net/http/
